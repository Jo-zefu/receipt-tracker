# Web 应用国际化 (i18n)

Web 应用已集成 **i18next** 国际化支持，自动检测浏览器语言设置。

## ✨ 功能特性

### 自动语言检测
- 检测浏览器语言设置（`navigator.language`）
- 中文用户自动显示中文界面
- 英文用户自动显示英文界面
- 语言偏好保存在 localStorage

### 支持的语言

- **English (en)** - 默认语言
- **简体中文 (zh)** - 中文界面

## 📝 已翻译的页面

### 1. 导航栏
- Logo: ReceiptScan / 票据扫描

## 🔧 技术实现

### 依赖包
```json
{
  "i18next": "^26.3.6",
  "react-i18next": "^17.0.9",
  "i18next-browser-languagedetector": "^8.2.1"
}
```

### 配置文件
`src/i18n.ts` - 包含所有翻译文本和配置

### 使用方式
```tsx
import { useTranslation } from 'react-i18next'

function MyComponent() {
  const { t } = useTranslation()
  
  return <h1>{t('receipts.title')}</h1>
}
```

## 🌍 语言检测规则

1. **浏览器语言设置**（navigator.language）
   - `zh`, `zh-CN`, `zh-TW` → 中文
   - 其他 → 英文

2. **LocalStorage 缓存**
   - 用户选择的语言会保存到 `localStorage`
   - 下次访问自动使用上次的语言

3. **Fallback 机制**
   - 如果某个翻译键缺失，自动回退到英文

## 📦 翻译文本结构

## 🚀 测试语言切换

### 方法 1: 浏览器语言设置
1. Chrome: 设置 → 语言 → 添加中文并设为首选
2. 刷新页面，界面自动切换为中文

### 方法 2: 浏览器控制台
```javascript
// 切换到中文
localStorage.setItem('i18nextLng', 'zh')
location.reload()

// 切换到英文
localStorage.setItem('i18nextLng', 'en')
location.reload()
```

## 📝 添加新翻译

1. 打开 `src/i18n.ts`
2. 在 `en.translation` 和 `zh.translation` 中添加新键值对
3. 在组件中使用 `t('your.new.key')`

示例：
```typescript
// src/i18n.ts
{
  en: {
    translation: {
      'settings.title': 'Settings'  // 新增
    }
  },
  zh: {
    translation: {
      'settings.title': '设置'  // 新增
    }
  }
}

// 组件中使用
<h1>{t('settings.title')}</h1>
```
